# linuxremote

 打包
 dpkg -b pack/ remoteapp.deb
 
 安装
 dpkg -i remoteapp.deb
 
 卸载
 dpkg -r remoteapp.deb
 
创建桌面快捷方式

复制 /usr/share/applications/remoteapp.destop 到桌面
右键点击允许启动
